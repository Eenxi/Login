# Login
本项目使用IDEA，利用Maven管理项目，整合SSM框架,实现简单登录注册功能
<h2>使用步骤：</h2><br/>
1.下载项目，导入idea编辑器。<br/>
2.下载sql数据库文件，导入到你的数据库中。<br/>
3.运行，不要忘了将jdbc.properties中的数据库密码和数据库名改为你自己的。<br/>
